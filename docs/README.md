## 기능 목록 
- [x] 다리 길이를 입력받는다.
  - 예외 처리 : 숫자 이외 문자를 포함한 입력, 정수 범위 밖 입력, 다리 길이 범위 밖 입력
- [x] 다리를 생성한다.
- [x] 이동할 칸을 입력받는다.
  - 예외 처리 : U, D가 아닌 입력
- [x] 이동 결과를 출력한다.
- [x] 건널 수 없는 칸으로 이동하거나 다리 끝까지 이동할 때까지 반복해서 입력받는다.
- [x] 건널 수 없는 칸으로 이동하면 게임을 다시 시도할지 여부를 입력받는다.
  - 예외 처리 : R, Q가 아닌 입력
  - 재시작해도 처음에 만든 다리로 재사용한다.
- [x] 다리를 끝까지 건너면 게임을 종료한다.
- [ ] 최종 게임 결과를 출력한다.
  - 게임 성공 여부와 시도 횟수를 포함한다.
